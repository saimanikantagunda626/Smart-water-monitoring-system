Project Description

The Smart Water Monitoring System is an advanced analytics platform designed to bridge the gap between urban consumption and sustainable management. By leveraging real-time data stored in MongoDB Atlas, the system provides municipal authorities with actionable insights to identify leakages, optimize distribution, and promote conservation.
